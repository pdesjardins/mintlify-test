By default, a Gift/Rewards Card Program restaurant group is created for every Toast management group. This group is used exclusively to manage restaurant locations that are part of your loyalty program. When you add a new location, if you are using the loyalty program, you should add it to this group (assuming the location is participating in the loyalty program). Note that the Gift/Rewards Card Program restaurant group does not appear in the You are viewing menu that appears in many locations throughout Toast Web because, other than adding locations to it, you cannot edit it. See [Filtering pages](adminGuide-filteringPagesForSpecificRestaurantGroupsAndLocations) for more information on the You are viewing menu.
