The currency value of guest credits remains available to a guest for a limited number of days. You can configure the number of days that credits remain available in the Credit expirationcontrol when you enable this feature for a restaurant. For more information about enabling and configuring the customer credit feature, see [Enabling the customer credits feature](docs/en-us/adminGuide-adminEnablingCustomerCredits).

Each time you give credits to a guest, you create a guest credits transaction. The value you add in each separate guest credits transaction remains available for the number of days set in the credit expiration configuration for the restaurant at the time that you created the transaction.
