### Standard API access requirements

The following are required to enable standard API access:

- You must be an active employee of that Toast location you are creating credentials for.


- An active subscription to [Toast Restaurant Management Suite (RMS) Essentials](https://www.toasttab.com/customers/shop/software-services/restaurant-management-suite?tier=Essentials)or higher. Sign in with your Toast Web credentials to be directed to Toast Shop where you can purchase RMS. RMS grants you access to the Toast Partner Integrationspage in Toast Web.


- You must have the Manage Integrationspermission for the locations you want to add standard API access credentials to. For more information, see [adminGuide#adminAccountAdminAccessPermissions].
