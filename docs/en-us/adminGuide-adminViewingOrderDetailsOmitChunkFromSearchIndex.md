### About the Orders and Order details reports

The Orders and Order details reports both provide detailed information about orders. They display the same information and use the same filters.

#### Orders report

The Orders report displays a tabular list of orders that match the filter criteria.

![Orders report with the filter fields and list of matching orders.](https://doc.toasttab.com/doc/media/orders-report.png)

When you click an order, the order details dialog is displayed. The order details dialog contains all of the details for the selected order.

![Order details dialog displayed from the Orders report.](https://doc.toasttab.com/doc/media/orders-report-order-details-dialog.png)

#### Order details report

The Order details report starts with a summary of the menu items from the orders that match the filter criteria.

![Menu item summary on the Order details report.](https://doc.toasttab.com/doc/media/order-details-menu-item-summary.png)

The menu item summary is followed by an order details panel for each matching order.

![Order details panel on the Order details report.](https://doc.toasttab.com/doc/media/order-details-order-panel.png)
