Toast Web allows you to copy configuration entities as a starting
    point for creating a new configuration entity. Copying configuration
    entities is a separate and distinct task from creating a version of an
    entity. When you copy a configuration entity, the copied entity has no
    relationship to the original entity beyond having the same values for the
    copied properties. The two configuration entities are treated completely
    separately for the purposes of reporting and editing.

Versioning, on the other hand, maintains a relationship between the
    different versions of a configuration entity. This allows Toast platform
    reports to consolidate information across versions and report on them as
    if they were all one entity.
