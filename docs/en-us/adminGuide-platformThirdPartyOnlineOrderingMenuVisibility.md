> **Note**
> 
> This feature is in limited release for DoorDash, Grubhub, and Uber Eats integrations. Third-party online ordering configuration pages are only accessible for the online ordering channels you have an integration with. If you do not have any integrations, you see a message directing you to Toast Shop where you can purchase online ordering integrations.


You can adjust menu visibility for an online ordering channel from the Third party ordering page in Toast Web. The Third party ordering page displays the following information:

- Third-party online ordering channel name


- Third-party online ordering channel toggle status


- Store name and location


- Third-party online ordering hours


- Third-party online ordering menu(s)



## Adding a menu to a third-party online ordering channel



> **Note**
> 
> Menus that are available on the Third party ordering page must have the Online ordering partners visibility setting enabled. This allows the menu to be visible to third-party online ordering channels. For more information on configuring your menu visibility, see [Complementary visibility settings on Third party ordering page](adminGuide-platformRestrictingMenuVisibilityToSpecificOnlineOrderingPartnersLegacy#platformComplementarySettingsOnTheThirdPartyOrderingPageLegacy).


**Procedure 3.23. To add a menu to a third-party online ordering channel**

1. [Access Toast Web ](adminGuide-adminAccessToastAdminBackend).


2. Choose Takeout & delivery &gt; Third party ordering. This displays all your third-party online ordering channels and their online ordering settings.


3. On the Third party ordering page, select the third-party online ordering channel you want to add a menu for. This allows guests to place orders from that third-party online ordering channel's menu.


4. Select Menu Settings to open the channel’s online ordering configuration page.


5. Select the Edit link to view the channel’s online ordering details. All available menus are listed under the Menus heading.


6. To add a menu, navigate to the Add Menudrop-down to select and add a menu.


7. Select the Add button. A `(Menu name) has been added `notification appears.


8. Select the Publish all changes button to save your changes. The Confirm Publish dialog box appears.


9. Select the Publish button to publish your changes or the Cancel button to cancel the dialog box. The menu is now available to be ordered from.


10. Repeat for each desired menu and third-party online ordering channel.



## Removing a menu from a third-party online ordering channel



> **Note**
> 
> You cannot bulk remove a menu from third-party online ordering channels on the Third party ordering page.


**Procedure 3.24. To remove a menu from a third-party online ordering channel**

1. On the channel’s online ordering configuration page, select the remove icon next to the menu to remove the menu from the channel.


2. A confirmation dialog box appears asking you if you want to remove the menu from the third-party online ordering channel.


3. Select the Remove button to remove the menu or the Cancel button to close out of the dialog box. A `(Menu name) has been removed` notification appears.


4. Select the Publish all changes button to save your changes. The Confirm Publish dialog box appears.


5. Select the Publish button to publish your changes or the Cancel button to cancel the dialog box.


6. Repeat for each desired menu and third-party online ordering channel.
