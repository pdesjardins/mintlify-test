### Advanced template examples

Because it is a more complex template to use, this section provides examples of different import operations when using the advanced template.



> **Note**
> 
> Due to their large size, the images in this section appear as thumbnails. To see the full-size image, right-click the image and choose Open image in new tab. You may also have to zoom in on the image after you open it in a new tab, to more easily see the details.


#### Menu item operations

This example creates a new menu item and attaches it to an existing menu group.

![Example of an import spreadsheet that creates a new menu item and attaches it to an existing menu group.](https://doc.toasttab.com/doc/media/menus-bulk-import-example-1.png)

This example attaches a new menu item to an existing menu group.

![Example of an import spreadsheet that attaches a new menu item to an existing menu group.](https://doc.toasttab.com/doc/media/menus-bulk-import-example-2.png)

#### Modifier group operations

This example creates a new modifier group and attaches it to a new menu item *in the same import file*.

![Example of an import spreadsheet that creates a new modifier group and attaches it to a new menu item in the same import file.](https://doc.toasttab.com/doc/media/menus-bulk-import-example-4.png)

This example creates a new modifier group and attaches it to an existing menu item.

![Example of an import spreadsheet that creates a new modifier group and attaches it to an existing menu item.](https://doc.toasttab.com/doc/media/menus-bulk-import-example-5.png)

This example attaches a new modifier group to a new menu item *in the same import file*.

![Example of an import spreadsheet that attaches a new modifier group to a new menu item in the same import file.](https://doc.toasttab.com/doc/media/menus-bulk-import-example-7.png)

This example creates a new modifier group and attaches it to two existing menu items, one in the `CREATE` row and the other in the `ATTACH` row.

![Example of an import spreadsheet that creates a new modifier group and attaches it to two existing menu items, one in the CREATE row and the other in the ATTACH row.](https://doc.toasttab.com/doc/media/menus-bulk-import-example-8-and-5.png)

This example attaches an existing modifier group to a new menu item.

![Example of an import spreadsheet that attaches an existing modifier group to a new menu item.](https://doc.toasttab.com/doc/media/menus-bulk-import-example-10.png)
