Use these start-to-finish recipes to get your integration implemented
  and live. Follow best practices and make sure you don't miss a step.

- **Your Toast integration **What it takes to get your integration live and running on the
        Toast platform.


- **Webhooks **Receive HTTP messages from the Toast platform to support your
        integration.


- **Cash reports **Report on cash transactions using information from the Toast
        platform.


- **CRM & guest engagement **Build a CRM & guest engagement integration with the Toast
        platform.


- **Data warehouse integrations **Get access to detailed information about restaurants, including
        menus, sales, payments, and employees.


- **Delivery dispatch **Build a delivery dispatch integration with the Toast
        platform.


- **Digital signage **Build a digital menu signage integration with the Toast
        platform.


- **Employee management **Build an employee management integration with the Toast
        platform.


- **Gift card integrations **Build a gift card integration with the Toast platform.


- **Inventory tracking **Get information about the availability of menu items for guest
        orders.


- **Labor reports **Build labor reports using information from the Toast
        platform.


- **Loyalty integration **Build a loyalty integration with the Toast platform.


- **Online ordering **Build a guest experience that sends orders to Toast platform
        restaurants.


- **Reservations **Build a reservation integration with the Toast platform.


- **Sales reports **Build sales reports using information from the Toast
        platform.


- **Security integration **Build a security integration using information from the Toast
        platform.


- **Shift scheduling **Build a shift scheduling integration with the Toast
        platform.
