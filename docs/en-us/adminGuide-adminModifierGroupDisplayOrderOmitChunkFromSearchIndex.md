### Modifier group display order overview

The Toast platform has a number of settings and conditions that affect the order it uses to display modifier groups for a menu item. None of these settings or conditions on their own determine the display order for modifier groups. Rather, it is the combination of these settings and conditions that determines the display order.



> **Important**
> 
> The order of modifier groups that you see on a menu group or menu item details page *does not fully reflect* the order of the modifier groups as they appear in your Toast product channels.


This section describes how the Toast platform determines modifier group display order. It contains the following sections:

- The [Settings and conditions that affect modifier group display order](adminSettingsAndConditionsThatAffectModifierGroupDisplay.html)section describes the settings and conditions themselves.


- The [Understanding modifier group display order](adminUnderstandingModifierGroupDisplay.html) section describes how the Toast platform uses these settings and conditions to determine modifier group display order.


- The [Ensuring important modifier groups display first](adminEnsuringImportantModifierGroupsDisplayFirst.html) section describes Toast support recommendations for using the Display Ordering Priority setting to ensure important modifier groups are displayed first.
