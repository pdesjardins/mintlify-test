### Specifying a custom schedule for a menu

You can use a menu's Availability settings to define a specific time period that a menu is available or a specific set of days that the menu is available. By default, a menu's availability is set to Always, meaning the menu is available during all hours that the restaurant is open.

For multi-location restaurants, the time period you specify is in each location's local time.



> **Note**
> 
> Custom schedules can only be set at the menu level. You cannot specify a custom schedule for menu groups or menu items. Also, you can only set one time period per menu.


**Procedure 8.28. To specify a custom schedule for a menu**

1. In the Availability section of the menu's details page, select Specific days and time. A table appears that shows the days and time that the menu will be available.


2. To edit the availability table, select the edit icon (pencil) on the table row. You see the Edit days and timedialog.


3. In the Day(s) available section, select the days you want the menu to be available.


4. In the Time available section, the Available at all times option is selected by default, indicating that the menu is available during all hours that the restaurant is open on the days you selected in the previous step. To set specific hours that the menu is available, uncheck Available at all times to see the Start and End time settings, then specify start and end times.


5. Select Save to save your custom schedule and return to the menu details page.


6. Select Save again.


7. If you are ready for guests and employees to see these menu changes, select Publish all changes, then select Publish again in the Confirm Publish dialog. For more information, see [Understanding when to publish your menu](adminUnderstandingWhenToPublishYourMenu.html).
