The Toast platform uses the following hardware components.

- A personal or laptop computer which you use to access
          configuration settings and reports through the Toast Web
          website.


- Touchscreen (Android) tablets, which run the mobile app and
          are located throughout the restaurant. Collectively known as Toast
          POS devices, tablets running the Toast POS mobile app can be used as
          order terminals at server stations, customer kiosks, or on wireless
          hand-held devices; as kitchen display system (KDS) devices; and as
          payment terminals.


- Printers, which can include both thermal receipt printers and
          impact kitchen printers.



In addition to the tablet and printer hardware, the Toast platform
      can also include hardware peripherals such as scales, cash drawers, pole
      displays, and scanners.
