Location-specific pricing allows you to specify different prices for
    a menu item based on location or restaurant group. The location-specific
    pricing feature eliminates the need to create additional versions of a
    menu item solely for the purpose of specifying different prices for
    different locations. To use location-specific pricing, you create a single
    version of the menu item and then set the location-specific prices on that
    version. For example, you could set a corporate price of $10 for the menu
    item and then override that price for a specific location or group of
    locations. The location-specific pricing feature is the preferred method
    for setting prices per location or restaurant group (as opposed to
    creating versions of menu items with different prices for each location or
    group).
