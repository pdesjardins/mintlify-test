The content on the order details dialog in the Orders report is the same as the content on the order details panels in the Order details report.

The order details information starts with an order summary that includes the number of guests, number of checks, the source of the order, and the devices that were used to process the order.

This is followed by an entry for each check. For each check the, the order details contains:

- A summary that includes the server and the check and payment amounts


- Takeout and delivery details


- The list of menu item selections, including quantities, prices, and item-level discounts


- Service charges on the check


- Discounts on the check


- Payments on the check
