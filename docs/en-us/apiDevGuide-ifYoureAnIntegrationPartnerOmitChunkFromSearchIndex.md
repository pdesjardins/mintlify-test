## If you're an integration partner

### Partner integration overview

To extend the Toast platform with your company's technology, you can build a partner integration using Toast APIs. Your API access is granted through the Toast Partner Integrations program, which includes a formal application and certification process. Once approved, you receive scoped API access that enables you to read and write data on behalf of customers that your company shares with Toast. As an integration partner, you're part of an ecosystem that allows Toast to offer a wide range of third-party capabilities.
