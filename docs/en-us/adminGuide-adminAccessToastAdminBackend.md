You use Toast Web to review and change configuration settings for your restaurant. Toast Web is a website that you sign into with your Toast account credentials. For example, you can edit menus, add restaurant employees, and view reports and analytics from Toast Web.

To access Toast Web, you must have email address-password user credentials and one or more of the Restaurant Adminaccess permissions. You can access Toast Web from a web browser or from the Toast POS app.

- From a browser, go to the [https://pos.toasttab.com](https://pos.toasttab.com)web site and select Login to access the Toast Web login page. Enter your email and password and then select Log In.


- From the Toast POS app, select the Toast logo until you reach the Toast POS home screen, and then select Setup \> Other Setup.



After you sign in, your assigned access permissions determine the set of options that appear. For more information, see the [Access Permission Reference](adminGuide-adminPermissions).
