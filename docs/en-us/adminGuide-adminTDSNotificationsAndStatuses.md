When an order is being delivered by a driver, the guest receives the
    following text notifications during the delivery process:

- Initial notification: This message is sent
        when the driver is en-route to the restaurant. The message contains a
        TDS tracking link that the guest can select to view driver information
        and location.


- Pick-up notification: This message is sent
        when the driver has picked up the guest's order.


- Rating notification: This message is sent
        when the guest's order has been marked as delivered. There is an
        option to rate the delivery experience.



For more information about TDS notifications, see [Do
    customers receive notifications for Toast Delivery Services
    orders?](https://central.toasttab.com/s/article/Do-customers-receive-notifications-for-Toast-Delivery-Service-orders?utm_campaign=toast-community&utm_medium=toastweb&utm_source=web)

## TDS statuses on Orders Hub

The Orders Hub displays various statuses that show the fulfillment
      progress for a TDS order. The Orders Hub displays the following statuses
      during the fulfillment and delivery process:

- DRIVER REQUESTED


- EN ROUTE TO RESTAURANT - ETA HH:MM


- HERE FOR PICKUP


- DRIVING TO DROP OFF


- DROPPING OFF - ETA HH:MM


- DELIVERED


- DELIVERY CANCELLED


- UNDELIVERABLE



For more information about TDS statuses, see [Toast Delivery Services statuses](adminGuide-platformManagingDeliveryOrders#platformTDSStatusReference).
