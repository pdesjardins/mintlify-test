For customers that use a Delphi digital menu board, Toast typically
    recommends creating a menu specifically for display on the digital menu
    board. If you have an existing menu that you want to display on the Delphi
    digital menu board, you can do so. In either case, to set up a menu for
    display on your Delphi digital menu board, you must configure the menu
    correctly in Toast Web and also coordinate with your Delphi onboarding
    consultant. For detailed information, see this [Toast
    Central article](https://central.toasttab.com/s/article/Create-a-Menu-for-the-Delphi-Digital-Menu-Board).
