- Manual entry on the Toast POS: The gift card identifier can be up to 21 numeric characters.


- Gift card API transactions: The gift card identifier can be up to 100 alphanumeric characters.


- QR code scanning: The gift card identifier can be up to 100 alphanumeric characters.



For information about card number encoding requirements, see [Gift Card Encoding Requirements FAQ](https://central.toasttab.com/s/article/Gift-Card-Encoding-Requirements-FAQ-1492810278469).
