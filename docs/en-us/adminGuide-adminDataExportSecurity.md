The contents of the data export file directory for your restaurant
  group can only be accessed with the access key and secret key that you have
  been given. You cannot view data export files for any other restaurant group
  and no other restaurant group can view your data export files.

If your believe that an unauthorized user has access to your secret
  key, Toast support can revoke the key and issue you a new one.
